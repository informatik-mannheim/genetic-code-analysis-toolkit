Visualizes which codons of a given code are in the maximal self-complementary C3 codes.

Parameter
1st:Number of one of the 2016 maximal self-complementary C3-codes from the list, i.e. a number between 1 and 216.

Example for C3(1):
Input sequence:ACG CAC ACG AUU AUU
Analysis result:ACG CAC ACG AUU AUU
