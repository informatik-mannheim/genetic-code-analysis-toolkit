Determines the number of common codons of the code given and of one of the 216 maximal self-complementary C3 codes chosen.

Parameter
1st:Number of one of the 2016 maximal self-complementary C3-codes from the list, i.e. a number between 1 and 216.

Example for C3(1):
Input sequence:ACG CAC ACG AUU AUU
Analysis result:    in C3(1): 4

not in C3(1): 1
