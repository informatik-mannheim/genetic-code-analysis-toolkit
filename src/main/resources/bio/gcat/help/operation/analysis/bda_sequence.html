Visualizes, which tuples (codons) of a code given are in wich dichotomic class created by a binary dichotomic algorithm chosen. Currently, only codons (i.e. tuples of length 3) are allowed.

Parameter
The BDA as defined in the BDA Tool is chosen.

Example for the Rumer's partition:
Input sequence:ACG CUC UGU UCG UUU GCG
Analysis result:ACG CUC UGU UCG UUU GCG
