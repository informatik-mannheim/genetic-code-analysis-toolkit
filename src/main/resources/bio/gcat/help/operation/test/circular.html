Checks whether a given code has the property of n-circularity.

Definition: Let n ⊆ ℕ and X ⊆ B³ be a trinucleotide code. We say that X is a trinucleotide n-circular code if for any concatenation x₁...xm; of m ≤ n codons from X, there is only one partition into codons from X when read on a circle, i.e. the next letter after the last letter being the first letter.

Example 2-circular:
Input sequence:ACG GTA CGT CGG TAC
Test result:Pass

Example 3-circular:
Input sequence:ACG GTA CGT CGG TAC
Test result:Fail
