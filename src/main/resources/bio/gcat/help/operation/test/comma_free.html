Checks whether a given code has the property of comma-freeness.

Definition: A trinucleotide code X ⊆ B³ is called a comma-free code provided that given any two codons x₁, x₂ ∈ X any sub-codon of the concatenation x₁x₂ except for x₁, x₂ themselves does not belong to X.

Example:
Input sequence:AGU UGA UCA ACU CGA
Test result:Pass

Example:
Input sequence:AAA UGA UCA ACU CGA
Test result:Fail
