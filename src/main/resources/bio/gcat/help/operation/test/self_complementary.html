Checks the property of the self-complementarity of a code, i.e. if with each tuple its complementary tuple is also in the code.

The complementary tuple for a given tuple is defined as a result of the substitution of the nucleotide bases the way given below and afterwards the reversing of the result.
AdeninewithUracil
UracilwithAdenine
GuaninewithCytosine
CytosinewithGuanine

Example:
Input sequence:AGU UGA UCA ACU UCG CGA
Test result:Pass

Example:
Input sequence:AGU UGA UAC GCA CAG
Test result:Fail
