The "Find & Replace" functionality can be used to search for certain patterns in a sequence and replacing them with any other value.

How to search for patterns and all options of the Find & Replace dialog are explained in the help article about Find.

Choosing "Replace" will replace the currently selected match, by the replacement text ("Replace with") and search the sequence for the next match. "Replace All" replaces all matches in the sequence by the replacement text provided.

If capturing groups (see Find) are used, you can reference them in the replace text, by using $CAPTURING_GROUP_NUMBER, e.g. $1 for the first capturing group. This for example allows you to change the order of bases in a sequence string. Searching for "(.)A(.)" and replacing it with "$2A$1" will reverse the order of all codons in a sequence, which have an A in their middle position.

Find "(.)G(U|G)" and Replace "$2G$1":
Input sequence:AGU AGA AGG UAC GCA CAG
Output sequence:UGA AGA GGA UAC GCA CAG
