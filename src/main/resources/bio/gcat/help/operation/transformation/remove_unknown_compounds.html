This operation removes tuples which contain unknown symbols or are not valid (compare tessera).

Unknown tuples are displayed as UNK in the compound display. For instance codons (tuples of length three), unknown tuples are those not coding an amino acid, for tessera (tuples of length four), all valid 4-nucleotides which are not valid tessera (nonsense) will be removed.

Example:
Input sequence:ACA AAX XCU UAU GAG
Output sequence:ACA UAU GAG
