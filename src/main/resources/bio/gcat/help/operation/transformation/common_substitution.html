Applies one of the following bijective transformations of bases (permutations). When a transformation chosen, all nucleotide bases in the sequence will be substituted the following way:

id Transformation:
AdeninewithAdenine
CytosinewithCytosine
GuaninewithGuanine
UracilwithUracil

c Transformation (Π(CG)(AU)):
AdeninewithUracil
CytosinewithGuanine
GuaninewithCytosine
UracilwithAdenine

p Transformation (Π(AG)(CT)):
AdeninewithGuanine
CytosinewithUracil
GuaninewithAdenine
UracilwithCytosine

r Transformation (Π(AC)(GU), Rumer):
AdeninewithCytosine
CytosinewithAdenine
GuaninewithUracil
UracilwithGuanine

Π Transformations: Swap the specified nucleobases for all given tuples.

Π: {A,C,G,U} -> {A,C,G,U}.

Example c Transformation:
Input sequence:AGU UGA UAC GCA CAG
Output sequence:UCA ACU AUG CGU GUC
